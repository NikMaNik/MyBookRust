## Bool

В `Rust` ***bool type*** имеет два возможных значения:
1. True
2. False
  
Значения ***Bool type*** имеют размер в один байт;  
Логический тип в `Rust` задается с помощью `bool`.
```rust
#fn main() {
    let b:bool = true;
    println!("{}", b);
    let b:bool = false;
    println!("{}", b);
#}
```
